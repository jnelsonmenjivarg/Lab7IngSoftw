# GestorUTL - Componentes y Servicios

Mini-aplicación web en entorno de consola desarrollada sobre Node.js para demostrar la implementación práctica de la Ingeniería de Software Basada en Componentes (CBSE) y la Arquitectura Orientada a Servicios (SOA).

## Estructura del Proyecto

- `components/`: Contiene unidades de software de desarrollo propio encapsuladas.
- `services/`: Capa orientada al consumo e integración de microservicios e interfaces REST.
- `utils/`: Módulos utilitarios y envoltorios de componentes externos de npm.

## Instrucciones de Instalación y Uso

1. Clonar el repositorio.
2. Instalar las dependencias de node:
   ```bash
   npm install
   ```
